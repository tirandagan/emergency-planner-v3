# Emergency Planner - AI Documentation

This repository contains AI-assisted development documentation, planning materials, and architectural decisions for the Emergency Planner project.

## Contents

- **`PRD.md`** - Product Requirements Document
- **`tasks/`** - Task-specific documentation and decisions
- **`diagrams/`** - System architecture and flow diagrams
- **`refs/`** - Reference materials and research
- **`scratch/`** - Working notes and experimental ideas
- **`*_templates/`** - Templates for various development phases

**Note:** Preparatory materials (master idea, setup notes, brand package, etc.) are maintained in a separate private repository.

## Purpose

This documentation suite is designed to:
- Provide AI assistants with comprehensive project context
- Track architectural decisions and rationale
- Maintain development roadmaps and task breakdowns
- Store reusable templates for consistent workflows

## Usage

These documents are optimized for AI-assisted development workflows and serve as a knowledge base for understanding project requirements, architecture, and implementation decisions.
