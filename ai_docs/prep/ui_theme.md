### Background Color System

**Primary Background Color:**
- **Light Mode:** Pure white `0 0% 100%` (`--background`)
- **Dark Mode:** Blue-tinted dark `220 15% 8%` (`--background`)
- **Usage:** Main content areas, landing page sections, chat interfaces, primary message areas

**Secondary Background Color:**
- **Light Mode:** Light gray `240 4.8% 95.9%` (`--muted`)
- **Dark Mode:** Blue-tinted muted `220 12% 15%` (`--muted`)
- **Usage:** Sidebars, alternating sections, secondary content areas, card backgrounds

**Design Strategy:** The two-color background system creates visual hierarchy and section differentiation across all application interfaces. Light mode uses subtle grays for depth, while dark mode uses custom blue-tinted neutrals that reinforce brand identity and create a cohesive, professional atmosphere.
